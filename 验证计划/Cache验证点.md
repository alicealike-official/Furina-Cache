## 一、功能验证

| 验证点 | 验证策略 | 验证方法 |
|--------|----------|----------|
| **读命中** | 先写入特定地址使其缓存，再读取相同地址 | 定向测试 + 参考模型比对 |
| **读未命中** | 访问未缓存地址，触发内存读取 | 定向测试 + 内存模型比对 |
| **写命中** | 写已缓存地址，更新缓存数据并置脏 | 定向测试 + 内部状态检查 |
| **写未命中** | 写未缓存地址，触发写分配和替换 | 定向测试 + 替换策略验证 |
| **字节/半字写** | 使用字节掩码进行部分写操作 | 定向测试 + 数据比对 |
| **读后写一致性** | 同一地址读写交替进行 | 随机测试 + Scoreboard比对 |
| **写后读一致性** | 写操作后立即读同一地址 | 随机测试 + 数据比对 |
